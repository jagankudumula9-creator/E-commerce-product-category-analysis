# 🛒 E-Commerce Product Category Analysis

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on e-commerce product data to uncover insights about categories, pricing, and ratings.

---

## 📊 Key Insights

* Electronics category dominates sales
* Most products fall under mid-price range
* Ratings are generally above 4.0
